Diagram 02
Three-Layer Defense Stack
Three independent security checks run in parallel on every telemetry tick. Any single layer confirming an attack triggers instant permanent lockdown.
Defense Architecture — Three Orthogonal Dimensions of Identity
01

Behavioral ML — Dense Autoencoder

Trained on 20,400 real keystroke samples from the CMU Keystroke Dynamics Benchmark (51 subjects × 400 attempts). Learns your typing rhythm in 5 seconds and flags anything it cannot reconstruct.

Detection signal: MSE > personal_threshold → behavioral anomaly detected
Catches: Credential theft · account sharing · bots · scripted replay attacks

↓    parallel    ↓
02

WebGL Hardware Fingerprint

Every GPU renders a WebGL triangle differently at the hardware level due to differences in anti-aliasing, floating-point precision, and color interpolation. The pixel output is hashed using cyrb53 and stored on first connection.

Detection signal: incoming gpu_hash ≠ baseline gpu_hash → different physical machine detected
Catches: Session cookie theft · token replay attacks · machine-in-the-middle

↓    parallel    ↓
03

Geovelocity Engine

Calculates the great-circle distance between consecutive IP addresses using the Haversine formula. Divides by elapsed time to derive implied travel speed. ASN check also flags datacenter and proxy IPs.

Detection signal: speed > 1,000 km/h → physically impossible travel
Catches: VPN abuse · proxy tunneling · hijacked accounts accessed from remote locations

The Six Behavioral Features
⏱️
mean_hold
Average duration each key is pressed. Reflects typing speed and confidence. Slow typists hold longer.
✓ Real CMU data
📊
std_hold
Consistency of hold times. A 10-year typist is metronomic (low std). An attacker under pressure is erratic (high std).
✓ Real CMU data
✈️
mean_flight
Average gap between releasing one key and pressing the next. The primary typing speed signal.
✓ Real CMU data
🎵
std_flight
Rhythm regularity across key transitions. Subconcious and nearly impossible to fake at typing speed.
✓ Real CMU data
🖱️
mouse_dist
Total pixel distance traveled per second. Captures motor control range and cursor habits.
~ Synthesized (live)
⌫
error_rate
Backspace frequency. Reflects typing confidence and cognitive load. Spikes under stress or unfamiliarity.
~ Synthesized (live)

⬆ Score Increases — Threat Events

Behavioral anomaly (per 1-second tick)+5 to +15
WebGL hardware fingerprint mismatch+100 instant
Impossible travel speed detected+100 instant
Datacenter / proxy IP detected+100 instant

⬇ Score Decreases — Clean Ticks

Clean tick (MSE below personal threshold)−6 per tick
Score floormin 0

Recovery intentionally slower than attack. 12 clean ticks to reverse 5 bad ones. Irreversible latch prevents pause-and-wait bypass.
